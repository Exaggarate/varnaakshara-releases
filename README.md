# Varnaakshara IME

**Indian Script Input Method Editor for Windows**

Type in English → Get Indian script. Works in any application.

## Supported Languages (12)
Assamese · Bengali · Gujarati · Hindi · Kannada · Malayalam · Marathi · Odia · Punjabi · Sanskrit · Tamil · Telugu

## Download
Go to [**Releases**](https://github.com/Exaggarate/varnaakshara-releases/releases) and download `Varnaakshara.exe`.

Just run it — no installation, no Python, no dependencies needed.

## How It Works
Varnaakshara uses the **Baraha phonetic transliteration** scheme. Type how the word sounds in English, and it converts to the target Indian script in real-time.

### Examples
| You type | Hindi | Kannada | Tamil |
|----------|-------|---------|-------|
| `namaste` | नमस्ते | ನಮಸ್ತೆ | நமஸ்தெ |
| `bhArat` | भारत | ಭಾರತ | பாரத |
| `dharm` | धर्म | ಧರ್ಮ್ | தர்ம் |

## Features
- 🖥️ System tray app — runs in background, works in any app
- 🔤 Baraha-compatible phonetic transliteration
- 🔔 Toast notifications on language switch
- 📝 Implicit schwa for Hindi/Marathi/Bengali (natural word endings)
- 🕉️ Proper Sanskrit halant handling
- 🌐 Candra vowels for Hindi loanwords (ऑफ़िस, डॉक्टर)
