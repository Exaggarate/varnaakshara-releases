# Varnaakshara IME — Indian Script Input Method Editor

**Type in 12+ Indian languages using simple English transliteration.**

Varnaakshara IME is a free, lightweight system tray application for Windows that lets you type in Indian scripts (Devanagari, Kannada, Telugu, Tamil, and more) using intuitive Baraha-style English-to-Indic transliteration. Works in any application — Word, Notepad, Chrome, WhatsApp Desktop, everywhere.

## ✨ Features

- **System-wide IME** — works in any Windows application
- **Baraha-style transliteration** — type English, get Indic script
- **12 languages** with instant F11/F12 switching
- **Smart implicit schwa** handling (Hindi-family languages)
- **Native script digits** — 0-9 outputs native numerals when active
- **Candra vowels** — `~e` → ऍ/ॅ, `~o` → ऑ/ॉ
- **Avagraha** — `&` → ऽ
- **Danda/Double danda** — `|` → ।, `||` → ॥
- **Auto-updater** — one-click updates from the tray menu
- **Lightweight** — sits in system tray, minimal resource usage
- **No internet required** — fully offline after installation

### Installation
1. Download the installer from [Releases](https://github.com/Exaggarate/varnaakshara-releases/releases)
2. Run the installer (SmartScreen warning: click **More info** → **Run anyway** — the app is not code-signed yet)
3. Launch from Start Menu or Desktop shortcut
4. The app appears in your system tray — right-click for options
5. Press **F11/F12** to switch between English and Indic scripts
